# Markdown Cheat Sheet

Headings:
```
# H1
## H2
### H3
```

Text formatting:
```
**bold text**
```
```
*italicized text*
```
```
> blockquote
```

Lists:
```
1. First item
2. Second item
3. Third item
```
```
- First item
- Second item
- Third item
```

Code:
```
`inline code`
```
````
```
{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}
```
````

Links:
```
[title](https://www.example.com)
```
Images:
```
![alt text](image.jpg)
```
